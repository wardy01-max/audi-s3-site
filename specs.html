The Engine Story

From 1.8T to mHEV — One Lineage

The displacement may have grown, the technology may have transformed, but every S3 engine shares the same soul: a turbocharged four-cylinder tuned to deliver its power in a way that feels immediate, characterful, and relentlessly quick.

First Generation

Audi S3 8L

1999 – 2003

Engine code BAM / AMK
Layout Inline-4, transverse
Displacement 1,781 cc
Compression ratio 9.5:1
Max power 210 hp @ 5,900 rpm
Max torque 270 Nm @ 1,950–5,000 rpm
Turbocharger KKK K03S single
Fuel system Multi-point injection
Transmission 6-speed manual (0A4)
AWD system Haldex Gen 1 / Gen 2
Front suspension MacPherson strut
Rear suspension Torsion beam
Front brakes 312 mm ventilated disc (Brembo)
Rear brakes 245 mm solid disc
Length / Width / Height 4,160 / 1,735 / 1,418 mm
Kerb weight ~1,395 kg
Boot volume 230 litres

Second Generation

Audi S3 8P

2006 – 2012

Engine code BWA / CDLB / CDLA
Layout Inline-4, transverse
Displacement 1,984 cc
Compression ratio 10.5:1
Max power 265 hp @ 6,000 rpm (268 hp facelift)
Max torque 350 Nm @ 2,500–5,000 rpm
Turbocharger BorgWarner K04 single
Fuel system FSI direct injection
Transmission 6-spd Manual or S tronic DSG
AWD system Haldex Gen 4
Front suspension MacPherson strut
Rear suspension Multi-link
Front brakes 340 mm ventilated disc
Rear brakes 310 mm ventilated disc
Length / Width / Height 4,224 / 1,765 / 1,421 mm
Kerb weight ~1,440 kg
Boot volume 250 litres (Sportback: 340 L)

Third Generation

Audi S3 8V

2013 – 2020

Engine code CJXB / CJXA / DJHB
Layout Inline-4, transverse
Displacement 1,984 cc
Compression ratio 9.6:1
Max power 296 hp (306 hp from 2016)
Max torque 380 Nm @ 1,800–5,500 rpm
Turbocharger IHI IS38 single
Fuel system TFSI combined port & direct injection
Transmission S tronic 6 (early) / S tronic 7
AWD system Haldex Gen 5 with e-diff simulation
Front suspension MacPherson strut
Rear suspension Multi-link
Front brakes 340 mm ventilated disc
Rear brakes 310 mm ventilated disc
Length / Width / Height 4,288 / 1,798 / 1,413 mm (Saloon)
Kerb weight ~1,385 kg (Sportback)
Boot volume 280 L (Sportback) / 425 L (Saloon)

Fourth Generation

Audi S3 8Y

2020 – Present

Engine code DNUE / DKZC
Layout Inline-4, transverse
Displacement 1,984 cc
Compression ratio 10.5:1
Max power 306 hp / 228 kW @ 5,000–6,500 rpm
Max torque 400 Nm @ 2,000–5,200 rpm
Turbocharger IHI IS38 (revised) single
Mild hybrid 12V belt-integrated starter-generator
Fuel system TFSI combined port & direct injection
Transmission S tronic 7 (DL382)
AWD system quattro with torque-vectoring rear diff
Front suspension MacPherson strut (progressive springs)
Rear suspension Multi-link
Front brakes 345 mm ventilated disc
Rear brakes 310 mm ventilated disc
Length / Width / Height 4,341 / 1,816 / 1,407 mm (Saloon)
Kerb weight ~1,460 kg (Sportback)
Boot volume 325 L (Sportback) / 430 L (Saloon)
